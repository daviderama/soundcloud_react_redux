# SoundCloud client built with React and Redux

1. `npm install`
2. `npm start`
3. visit `http://localhost:8080/`

### Stack

- Webpack
- webpack-dev-server
- Babel
- React
- react-hot-loader
- react-router
- Redux
- redux-logger
- react-redux
- react-router-redux
- redux-thunk
- soundcloud


### Test
- mocha
- chai
- jsdom
- Enyzme
